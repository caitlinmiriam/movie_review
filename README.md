🎬 WatchView
WatchView is a simple HTML-only movie and entertainment website that displays trending movies, TV shows, and provides basic navigation and contact features.

🌟 Features
---------------------------------
🎬 Home page with welcome section
🍿 Top movies and shows display
🧭 Simple navigation bar
📩 Contact page with email link (mailto:)
🖼️ Movie posters displayed in table format
📱 Beginner-friendly HTML structure

Navigate using the menu links
